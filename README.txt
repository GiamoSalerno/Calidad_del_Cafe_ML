En este repositorio se busca crear un modelo de machine learning que determine la calidad del café.

El repositorio consta de diferentes carpetas:
-data: Aquí se encuentran todos los datos utilizados en diferentes formas; en crudo, procesado, y subdividido en train y test para sus pruebas.
-src: En esta carpeta están los ejecutables (.py) donde se procesan los datos, se define el modelo y se evalúa su rendimiento.
-models: Se encuentran los modelos creados, además de un archivo para definir los parámetros a utilizar (.yaml)
-notebooks: Aquí se encuentra el análisis exploratorio y pruebas previas para el desarrollo de modelos.

De momento no se incluyen en el repositorio, pero además se encuentran carpetas como:
-app: en donde se preparará una aplicación de Streamlit para una presentación técnica.
-docs: se podrán encontrar presentaciones básicas para negocio.
